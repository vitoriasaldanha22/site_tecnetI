# Site TecnetI

## Descrição

Projeto de site institucional desenvolvido utilizando HTML para estruturação das páginas, voltado para apresentação de informações e serviços da TecnetI.

## Tecnologias Utilizadas

- HTML

## Funcionalidades

- Apresentação institucional
- Páginas informativas
- Estruturação semântica de conteúdo

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/vitoriasaldanha22/site_tecnetI.git
   ```
2. Abra o arquivo `index.html` no navegador.
